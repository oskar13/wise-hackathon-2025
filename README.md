## Pitch

Wise achievements introduce a fun layer on top of the app. Users unlock achievements by exploring different features of the app like ordering a debit card, making payments and donations.

To make achievements meaningful we introduce **Wise Coin** which can be used for all sorts of things on the platform like discounts for fees, exclusive merch, ordering custom designs for your card.
